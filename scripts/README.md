# Analysis codes and scripts

This directory contains the analysis codes and scripts used for the manuscript

## Contents

This directory contains the setup and general scripts run first before running the separate workflows in the subdirectories

The numbering of the scripts indicates running order (e.g. 0a, 0b, 1, 2, 3)

The purpose of each script is described within the script itself

Subdirectories for the separate workflows (numbering indicates running order):

- `1_read_based/`  
  Scripts for the read-based analysis run after the setup and general scripts, associated to figures 2-5 of the manuscript, supplementary figures 3-5 & 10-11 and supplementary tables 3-21
  
- `2_contig_based/`  
  Scripts for the contig-based analysis run after the read-based workflow, associated to supplementary figures 6-7 and required for the MAG-based workflow

- `3_mag_based/`  
  Scripts for the MAG-based analysis run after the contig-based workflow, associated to figure6 6 of the manuscript, supplementary figure 12 and supplementary tables 23-24

- `R_scripts/`  
  R scripts used by the read-based, contig-based and mag-based workflows and scripts for producing the plots, some results and related information for the manuscript
