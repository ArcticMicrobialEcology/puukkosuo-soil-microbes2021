# R scripts 

This directory contains the R scripts used by the setup and general scripts and sripts in the read-based, contig-based and mag-based worklows

The directory also contains R scripts for producing the figures, supplementary figures, supplementary tables, related results and information for the manuscript

## Contents

This directory contains the R scripts for the setup and general scripts run first before running the separate workflows

The numbering of the scripts indicates running order (e.g. 0a, 0b, 1, 2, 3)

The purpose of each script is described within the script itself

Subdirectories for the separate workflows and manuscript figures:

- `read_based/`  
  R scripts associated with read-based workflow and used by the scripts in `scripts/1_read_based/` and `scripts/2_contig_based/` for some downstream processing and analysis
  
- `contig_based/`  
  R scripts associated with contig-based workflow and used by the scripts in `scripts/2_contig_based/`  

- `mag_based/`  
  R scripts associated with MAG-based workflow and used by the scripts in `scripts/3_mag_based/`  

- `manuscript_figures/`  
  R scripts for producing the figures, supplementary figures and tables for the manuscript
