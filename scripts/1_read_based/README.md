# Analysis codes and scripts for the read-based analysis

This directory contains the analysis codes and scripts used for the read-based analysis for the manuscript

## Contents

This directory contains the scripts for the read-based analysis

The numbering of the scripts indicates running order (e.g. 0a, 0b, 1, 2, 3)

The purpose of each script is described within the script itself

- `downstream/`  
  This subdirectory contains scripts for processing and analysing the acquired and compiled read-based result data. Contains scripts for parsing and filtering the phyloflash taxonomic data, filtering the compiled metabolic marker gene and KEGG KO data, analyzing differential abundance and expression with linear mixed effect modelling in metagenomics and metatranscriptomics data, estimating KEGG module completeness and performing gene set enrichment analysis for the core KEGG metabolic modules
