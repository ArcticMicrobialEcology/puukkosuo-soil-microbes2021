# Puukkosuo study enviromental data

This directory contains environmental data used by the workflow.

The files in this directory are used by various scripts throughout the data analysis workflow.

## Contents

- `2021-09-12_Oulanka_flux.xlsx` – Processed methane fluxes for the sample plots measured at 12.9.2021
- `2021-09-20_Oulanka_flux.xlsx` – Processed methane fluxes for the sample plots measured at 20.9.2021
- `Huokosvesi_Puukkosuo_kesä2021.xlsx` – Pore water measurements for the sample plots for summer 2021 (september values used in this study)
- `Temperature_Snow_depth.xlsx` – The average recorded peat temperature information for the sample plots for winter 2020-2021 and the average snow depth for March 2021 (time of maximum snow depth)
