# Puukkosuo study metadata

This directory contains the study metadata and environmental data used by the workflow.

The files in this directory are used by various scripts throughout the data analysis workflow.

## Contents

- `Oulanka_ACAP_study_site.xlsx` – The experimental design for the study
- `Puukkosuo_vegetation clusters.xlsx` – Vegetation cluster information for the sample plots
- `selected_KEGG_modules.txt` – A list of the core KEGG metabolic modules investigated in the study
